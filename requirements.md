# Requirements Document
Updated functional requirements

## Functional Requirements
1. User can search books by title or author.
2. User can borrow books.
3. User can return books.
4. Admin can add new books.
5. Admin can remove books.

## Non-Functional Requirements
1. System should be fast (response time < 2 seconds).
2. System should be secure (login required).
3. System should be user-friendly.
4. System should be available 24/7.
5. System should handle multiple users at once.
